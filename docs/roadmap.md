# Roadmap

## Semana 1 — Definición y diseño base
- Definir especies iniciales (5–10)
- Recolectar información (nombre, tipo, descripción, peligrosidad)
- Definir estructura de la web (páginas y secciones)
- Crear wireframes simples (papel o digital)
- Definir paleta de colores básica

## Semana 2 — Estructura HTML
- Crear estructura base del proyecto
- Desarrollar páginas HTML (home, categorías, ficha)
- Implementar navegación entre páginas

## Semana 3 — Estilos (CSS)
- Aplicar estilos generales (colores, tipografías)
- Diseñar layout (flexbox/grid)
- Crear componentes visuales (cards de especies)

## Semana 4 — Lógica con JavaScript
- Crear archivo de datos (reptiles.js)
- Mostrar especies dinámicamente
- Implementar filtrado o búsqueda básica

## Semana 5 — Mejora y presentación
- Ajustes de diseño (responsive básico)
- Pruebas generales
- Mejorar experiencia de usuario
- Preparar README (capturas, descripción del proyecto)
- Subir versión final a GitHub
